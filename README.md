# eth-sheet

imports all ethereum txns to a google spreadsheet using covalent API

```
ETHEREUM_ADDRESSES=
COVALENT_API_KEY=
RPC_URL=
GOOGLE_SHEETS_API_KEY=
GOOGLE_API_EMAIL=
GOOGLE_API_PK=
REDIS_URI=
```

Defaults to fetching transactions from block 13907728 to current.

Expected to run via cron job.
